# Midicat

cat/tailf for MIDI input events

![demo screen](./demo.gif)

## Installation

Install it yourself as:

    $ gem install midicat

## Usage

Just execute

```
$ midicat
```

after installing it

note: you should have gem executables directory in `$PATH` (`gem env gempath` shows the path to append your `$PATH`)
